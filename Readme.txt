Приложение чат бот для сотрудников офиса( приватный чат для общения в коллективе)

Данное приложение было создано в связи с потребностью обмена информацией по работе в офисе, где отсутствует интернет и мобильная связь из-за проблем с безопасностью.

Простое в своём пользовании, решает вопрос связи внутри производства, сохраняет приватность внутри коллектива во время работы. 

Не имеет в себе уникальных свойств так, как создано решать конкретные задачи по обмену сообщениями во время часов работы предприятия.
